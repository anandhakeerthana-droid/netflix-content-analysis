# Netflix Content Analysis using Python

## Project Overview

This project explores the Netflix Movies and TV Shows dataset through Exploratory Data Analysis (EDA) using Python. The goal is to understand patterns in Netflix's content library by analyzing content types, countries, genres, ratings, content growth over time, and movie durations.

This project was completed as part of my journey in learning data analysis and demonstrates the complete EDA workflow, from data cleaning to visualization and insight generation.

---

## Objectives

- Explore Netflix's content catalog.
- Perform data cleaning and preprocessing.
- Visualize trends using different chart types.
- Generate meaningful business insights from the data.

---

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

---

## Analyses Performed

- Distribution of Movies vs TV Shows
- Countries contributing the most content
- Most common genres
- Netflix's content growth over time
- Distribution of content ratings
- Genre comparison between Movies and TV Shows
- Distribution of movie durations

---

## Key Findings

- Movies make up the majority of Netflix's catalog.
- The United States contributes the highest number of titles.
- International content is a significant part of Netflix's library.
- Netflix experienced rapid content growth until around 2019.
- Most content is rated TV-MA.
- Most movies have a duration between 90 and 120 minutes.
